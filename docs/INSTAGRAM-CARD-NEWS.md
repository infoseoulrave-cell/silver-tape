# 인스타그램 카드뉴스 (30종)

리서치 기반 콘텐츠로 제작한 인스타그램 피드용 카드뉴스입니다.

## 주제 (4가지)

1. **과거의 물감** — 라스코 벽화, 중세 팔레트, 유화·티리언 퍼플·인디고·조선 채색·튜브 발명 등
2. **디지털 vs 필름 카메라** — 원리, 장단점, 20~30대 필름 부활, 숫자로 보는 트렌드
3. **작품이 주는 효과** — 감정·평가, 병원·미술치료·촉각, 사무실 미술
4. **미술품 있는 집 인테리어** — 갤러리 인테리어, 저비용 고효과, 그림 선택·배치·아이 그림

## 포맷

- **비율:** 1080×1350 (4:5) — 2025 인스타그램 피드 권장
- **언어:** 한국어

## 사용 방법

1. **웹에서 보기**
   - `npm run dev` 후 브라우저에서 `/instagram/card-news` 접속
   - 목록에서 카드 클릭 → 단일 뷰에서 스크린샷

2. **PNG 일괄 추출**
   - 개발 서버 실행 중인 상태에서:
   - `node scripts/capture-card-news.mjs`
   - 필요: `npm install puppeteer`
   - 결과: `public/instagram/card-news/card-01.png` ~ `card-30.png`

3. **배경을 사진으로 바꾸기**
   - `src/data/card-news.ts`에서 각 항목에 `image: 'https://...'` 또는 `image: '/images/...'` 추가
   - 카드 단일 뷰에서 자동으로 배경 이미지 + 어두운 오버레이 + 흰 텍스트로 표시됨 (가독성 유지)

## 출처·리서치

- 라스코·동굴벽화 색소, 중세·르네상스 물감, 티리언 퍼플, 인디고, 조선 채색, 유화·튜브 발명
- 필름 vs 디지털 원리·장단점, 필름 부활·MZ·니콘 헤리티지
- 미술작품 감정·평가 연구, 병원·미술치료·코크란, 촉각 조형예술, 사무실 미술
- 갤러리 인테리어, 그림 선택·심리·공간별 배치
